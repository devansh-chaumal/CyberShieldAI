# 🛡️ CyberShield AI

An AI-powered phishing detection platform built using Python, Flask, HTML, CSS, and JavaScript.

---

## 📌 Project Overview

CyberShield AI is a cybersecurity web application that analyzes URLs to determine whether they are safe or potentially malicious.

The application combines traditional phishing detection techniques with AI-powered explanations to help users understand online threats.

## 🚀 Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Enter the project folder

```bash
cd CyberShieldAI
```

3. Create a virtual environment

```bash
python3 -m venv .venv
```

4. Activate the virtual environment

macOS/Linux:

```bash
source .venv/bin/activate
```

5. Install dependencies

```bash
pip install -r requirements.txt
```

6. Start the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
